# 地震勘探原理知识库

## 一、地震勘探概述

### 1.1 基本定义
地震勘探是一种地球物理勘探方法，通过人工激发地震波，研究地震波在地下岩层中的传播规律，从而查明地下地质构造和岩性分布，为油气、矿产资源和工程勘查提供依据。

### 1.2 工作原理
1. **激发**：在测线上选择激发点，用炸药、震源车等产生地震波
2. **传播**：地震波向地下传播，遇到不同岩性的地层界面时产生反射、折射和透射
3. **接收**：地面布置检波器接收返回的地震波信号
4. **记录**：地震仪记录地震波的旅行时、振幅、频率等信息
5. **处理与解释**：对记录数据进行处理，得到地震剖面，进行地质解释

## 二、地震波理论

### 2.1 地震波分类

#### 2.1.1 按传播方式分类
- **体波**：在介质内部传播的波
  - **纵波（P波）**：质点振动方向与波传播方向一致，传播速度最快
  - **横波（S波）**：质点振动方向与波传播方向垂直，速度约为P波的0.5-0.6倍
- **面波**：沿介质表面传播的波
  - **瑞利波**：质点作椭圆运动，振幅随深度衰减
  - **勒夫波**：质点作水平运动，只存在于层状介质中

#### 2.1.2 按用途分类
- **有效波**：能够反映地下地质信息的波，如反射波、折射波
- **干扰波**：干扰有效波识别和解释的波，如面波、多次波、随机噪声

### 2.2 地震波传播规律

#### 2.2.1 反射与透射
当地震波遇到波阻抗差异界面时，部分能量反射，部分能量透射：
- **反射定律**：入射角等于反射角
- **透射定律（斯涅尔定律）**：sinθ₁/v₁ = sinθ₂/v₂
- **反射系数**：R = (ρ₂v₂ - ρ₁v₁)/(ρ₂v₂ + ρ₁v₁)
- **透射系数**：T = 2ρ₁v₁/(ρ₂v₂ + ρ₁v₁)

#### 2.2.2 折射
当入射角等于临界角时，透射波沿界面滑行，产生折射波：
- **临界角**：θ_c = arcsin(v₁/v₂)
- **折射波**：在v₂ > v₁条件下产生，沿界面滑行后在介质1中返回地面

### 2.3 地震波特征参数
- **振幅**：反映波的能量大小，与反射系数、吸收衰减有关
- **频率**：反映波的振动快慢，不同地层对频率的滤波作用不同
- **相位**：反映波的振动状态，用于识别波的极性
- **波形**：反映波的形状特征，与震源子波和地层响应有关

## 三、野外数据采集

### 3.1 观测系统
观测系统是指激发点和接收点的空间排列关系，常用类型包括：

#### 3.1.1 二维观测系统
- **单边放炮**：炮点位于排列一端
- **中间放炮**：炮点位于排列中间
- **双边放炮**：炮点位于排列两端

#### 3.1.2 三维观测系统
- **线束状观测系统**：多条接收线平行排列
- **砖墙式观测系统**：接收点呈网格状分布
- **正交观测系统**：激发线和接收线正交

### 3.2 核心参数
- **道间距（Δx）**：相邻检波器之间的距离，通常10-50m
- **炮间距（Δs）**：相邻炮点之间的距离
- **偏移距**：炮点到最近检波器的距离
- **排列长度**：一次放炮接收排列的长度
- **覆盖次数**：地下同一反射点被不同炮检对观测的次数

### 3.3 仪器设备
- **震源**：炸药、气枪、震源车、可控震源等
- **检波器**：将机械振动转换为电信号的传感器
- **地震仪**：记录地震信号的仪器，包括采集站和主机
- **电缆**：连接检波器和采集站的传输线路

## 四、地震数据处理

### 4.1 预处理
- **格式转换**：将野外记录转换为处理系统可识别的格式
- **废道剔除**：剔除工作不正常的道
- **初至拾取**：拾取各道初至时间
- **静校正**：消除地形起伏和低速带影响

### 4.2 常规处理流程
1. **预处理** → 2. **反褶积** → 3. **速度分析** → 4. **动校正** → 5. **叠加** → 6. **偏移**

#### 4.2.1 反褶积
压缩震源子波，提高分辨率：
- **脉冲反褶积**：将地震子波压缩为尖脉冲
- **预测反褶积**：消除短周期多次波
- **子波整形**：将实际子波整形为零相位子波

#### 4.2.2 速度分析
求取地层速度参数：
- **速度谱**：显示不同速度对应的叠加能量
- **常速扫描**：用不同速度进行动校正叠加
- **层速度分析**：求取各层的层速度

#### 4.2.3 动校正（NMO）
消除炮检距对反射时间的影响：
- **NMO公式**：t(x) = √(t₀² + x²/v²)
- **NMO校正量**：Δt = t(x) - t₀
- **动校正拉伸**：远道校正量大，波形拉伸严重

#### 4.2.4 水平叠加
将不同炮检距的道按反射点叠加，提高信噪比：
- **叠加原理**：有效波同相叠加增强，随机噪声相互抵消
- **叠加次数**：n = N/(2Δx/Δs)，N为总道数
- **叠加效果**：信噪比提高√n倍

#### 4.2.5 偏移归位
将倾斜界面反射归位到真实空间位置：
- **绕射叠加偏移**：基于惠更斯原理
- **有限差分偏移**：求解波动方程
- **克希霍夫偏移**：积分法偏移
- **波动方程偏移**：最精确的偏移方法

### 4.3 特殊处理技术
- **AVO分析**：分析振幅随偏移距变化，识别岩性和流体
- **波阻抗反演**：将地震记录反演为波阻抗剖面
- **属性分析**：提取地震属性进行储层预测
- **多波多分量**：利用P波和S波信息进行综合解释

## 五、地震资料解释

### 5.1 构造解释
- **层位标定**：利用钻井、测井资料确定地震反射层的地质含义
- **断层解释**：识别断点、断面，确定断层性质
- **褶皱解释**：识别背斜、向斜等构造形态
- **不整合面解释**：识别角度不整合、平行不整合

### 5.2 岩性解释
- **振幅分析**：强振幅可能对应砂岩发育带
- **频率分析**：高频衰减可能指示含气层
- **波形分析**：波形变化反映岩性变化
- **速度分析**：速度异常可能指示特殊岩性

### 5.3 储层预测
- **储层识别**：利用地震属性识别储层发育区
- **厚度预测**：利用时差和速度预测储层厚度
- **物性预测**：利用地震属性预测孔隙度、渗透率
- **流体检测**：利用AVO、频率衰减等检测油气

## 六、特殊地震勘探方法

### 6.1 垂直地震剖面（VSP）
在井中放置检波器，地面放炮：
- **零偏VSP**：炮点位于井口
- **非零偏VSP**：炮点偏离井口
- **Walkaway VSP**：炮点沿测线移动
- **3D-VSP**：多炮点多检波器

### 6.2 井间地震
在一口井中放炮，另一口井中接收：
- **跨井地震**：研究井间地层
- **井间层析成像**：重建井间速度场
- **井间反射成像**：获得井间高分辨率剖面

### 6.3 多波多分量地震
同时接收P波和S波：
- **P-P波**：常规纵波
- **P-S波**：转换横波
- **S-S波**：纯横波
- **应用**：岩性识别、裂缝检测、流体识别

## 七、地震勘探应用领域

### 7.1 油气勘探
- **构造油气藏勘探**：寻找背斜、断层等构造圈闭
- **岩性油气藏勘探**：寻找砂岩透镜体、生物礁等岩性圈闭
- **地层油气藏勘探**：寻找不整合面上下地层圈闭

### 7.2 工程勘探
- **工程地质勘察**：查明地基条件
- **隧道勘探**：查明隧道围岩条件
- **地质灾害调查**：调查滑坡、崩塌等灾害体

### 7.3 矿产资源勘探
- **煤炭勘探**：查明煤层分布
- **金属矿勘探**：寻找控矿构造
- **地热资源勘探**：查明地热储层

## 八、计算公式

### 8.1 反射波旅行时
- **水平界面**：t₀ = 2H/v
- **倾斜界面**：t(x) = √(t₀² + x²/v² + 2t₀xsinφ/v)
- **弯曲界面**：需要数值计算方法

### 8.2 速度计算公式
- **平均速度**：v_avg = H/t
- **均方根速度**：v_rms = √(∑v_i²t_i/∑t_i)
- **层速度**：v_int = ΔH/Δt

### 8.3 深度计算公式
- **水平界面**：H = v·t₀/2
- **倾斜界面**：H = (v·t₀·cosφ)/2
- **弯曲界面**：需要迭代计算

## 九、重要概念解释

### 9.1 波阻抗
波阻抗是介质密度与波速的乘积：Z = ρ·v。波阻抗差异是产生反射波的根本原因。

### 9.2 反射系数
反射系数表示反射波振幅与入射波振幅之比：R = (Z₂ - Z₁)/(Z₂ + Z₁)。正反射系数表示反射波与入射波同相位，负反射系数表示反相位。

### 9.3 多次波
多次波是地震波在上下两个强反射界面之间多次反射形成的波。按路径可分为：
- **短程多次波**：在浅层界面间多次反射
- **长程多次波**：在深层界面间多次反射
- **鸣震**：在水层中多次反射

### 9.4 分辨率
地震分辨率是指区分两个相邻地质体的能力，包括：
- **垂向分辨率**：λ/4（λ为波长）
- **横向分辨率**：第一菲涅尔带半径：r = √(λH/2)

### 9.5 信噪比
信噪比是指有效信号与噪声的强度比。提高信噪比的方法包括：增加覆盖次数、滤波处理、叠加处理等。
